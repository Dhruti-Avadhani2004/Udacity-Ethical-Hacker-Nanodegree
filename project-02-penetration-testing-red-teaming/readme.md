## Project Overview

A full penetration test and red team engagement across multiple targets — a website, a Windows machine, and two Linux machines. The project involved end-to-end offensive security work: reconnaissance, scanning, exploitation, privilege escalation, and professional executive-grade reporting.

---

## Work Completed

**Reconnaissance**
- Researched the Learn About Security website
- Identified technologies used by the organisation
- Identified DNS records of the target site

**Windows Target**
- Scanned the Win-10 machine using Nmap
- Gained remote access by exploiting a vulnerable application on the device

**Linux Targets**
- Scanned `debianx64DMZOnCloudNew` using Nmap
- Scanned the `DMZi` machine using Nmap
- Gained remote access to `debianx64DMZOnCloudNew` using a **brute force attack** (Hydra)
- Gained remote access to `DMZi` by finding and using SSH keys

**Reporting Standards**
- Documented significant vulnerability details
- Captioned and centred all screenshots
- Ensured methodology was reproducible end-to-end
- Prioritised vulnerabilities by severity
- Provided executive-level recommendations
