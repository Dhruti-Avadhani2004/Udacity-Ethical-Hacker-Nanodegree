# Udacity Ethical Hacker Nanodegree

This repository documents the hands-on project work completed as part of the **Udacity Ethical Hacker Nanodegree**, completed on scholarship. It includes two comprehensive security assessment projects covering vulnerability management, penetration testing, OSINT, web application security, and red teaming.

---

## Program Overview

**Provider:** Udacity
**Program:** Ethical Hacker Nanodegree
**Mode:** Bertelsmann Tech Booster Scholarship
**Focus Areas:** Vulnerability assessment, penetration testing, OSINT, red teaming, security reporting

### Project 1 — Audit ExampleCorp
**Submitted:** April 22, 2025 | **Result:** ✅ Passed

A comprehensive security audit of a simulated corporate environment. Covered vulnerability scanning, exploitation of two critical CVEs, OSINT intelligence gathering, phishing analysis, and web application security testing.
