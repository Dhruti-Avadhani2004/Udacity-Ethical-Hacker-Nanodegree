# Project 1 — Audit ExampleCorp

**Program:** Udacity Ethical Hacker Nanodegree
**Submission Date:** April 22, 2025
**Result:** ✅ Passed

---

## Project Overview

A comprehensive security audit of a simulated corporate environment (ExampleCorp). The project covered the full lifecycle of a real-world security assessment — from initial reconnaissance and vulnerability scanning through to exploitation, documentation, and remediation recommendations.

---

## Work Completed

**Vulnerability Management**
- Calculated CVSS scores for discovered vulnerabilities
- Performed full vulnerability scan of the target machine using Nessus

**System Auditing**
- Exploited **CVE-2017-12635** and documented findings
- Exploited **CVE-2017-12636** and documented findings
- Performed network scan of the target using Nmap

**OSINT & Phishing**
- Reviewed OSINT data to identify compromising information
- Analysed phishing results and identified compromised accounts

**Application Audit**
- Exploited the target application using data gathered from OSINT and phishing analysis
